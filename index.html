Object of the Game


The object of the game is to correctly repeat the sequence. Depending on the game mode selected this will vary in complexity.


Press and Simon will give the first signal. Repeat the signal by pressing the same color button.


Classic Game Mode


Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color buttons, in order.


Simon will then duplicate these first two signals and add one.


Continue playing as long as you can repeat each sequence of signals correctly. After the 5th, 10th and 15th signals in a sequence, Simon automatically speeds up.


If you fail to repeat a sequence exactly then it's GAME OVER.
