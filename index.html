Instructions

Object of the Game


The object of the game is to correctly repeat the sequence. Depending on the game mode selected this will vary in complexity.


Press and Simon will give the first signal.

Classic Game Mode


Repeat the steps in the order they are presented, each round will increase in length.

Random Game Mode


Repeat the steps in the order they are presented, each round will be random and increase in length.

Last-Only Game Mode


Repeat the steps in the order they were presented, only the final step will play back but you will need to present the entire sequence, each round will increase in length.

Reverse Game Mode


Repeat the steps in a reverse order to which they are presented, each round will increase in length.


Continue playing as long as you can, repeat each sequence of signals correctly. After every 5th signal in a sequence, Simon automatically speeds up. If you fail to repeat a sequence exactly then it's GAME OVER.
